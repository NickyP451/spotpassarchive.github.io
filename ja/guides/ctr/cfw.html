SpotPass Archival Project / 3DS ガイド (CFW)

前提:

  • Luma3DS CFWが動作しているすべての改造済み2DS/3DS
    • CFWの有無を確認するには、システムの電源がオフの状態で、SELECTを押しながらPOWERボタンを押してください。すぐにトップ画面にテキストが表示されれば、CFWがあります。そうでない場合は、CFWがないことを意味しますので、戻ってCFW以外のオプションを選択する必要があります。
  • SDカード
  1. SpotPassDumper11をユニバーサルアップデータアプリからダウンロードしてください。 HOMEメニューから実行したい場合は .cia ファイルを、Homebrewランチャーから実行したい場合は .3dsx ファイルを選択してください。
    • Universal Updaterをお持ちでない場合は、 GitHub から直接ダウンロードすることもできます。
  2. HOMEメニューかHomebrew LauncherからSpotPassDumper11を開きます。
  3. ツールが動作するまで待ちます。下画面に情報が表示されます。
  4. お使いのシステムがインターネットに接続されている場合、SpotPassDumper11はあなたのSpotPassキャッシュを直接アップロードしようとします。キャッシュをインターネットにアップロードできなかった場合、partitionA.binファイルはSD/microSDカードの/spotpass_cache/フォルダに保存されます。
  5. あなたのシステムがダンプをアップロードできなかった場合、以下の緑色のリンクから送信することもできます。
