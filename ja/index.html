SpotPass Archival Project

ご存知の通り、任天堂はニンテンドー3DSソフトおよびWii Uソフトおけるオンラインプレイやいつの間通信など、4月9日で終了することを発表しました。

2024年4月9日（火）午前9時に、いつの間に通信につきまして、インターネット通信を用いた各種機能を終了ことになります。これに伴い、多くのタイトルいつの間に通信を利用した機能が利用できなくなります。

3DS/2DSおよびWii Uシステムには、最近いつの間に通信データを受け取ったタイトルのローカルデータベースが保存されています。このデータベースは、いつの間に通信コンテンツのダウンロードに使用するURLを探すために使用できます。

お使いのシステムからこのデータベースをバックアップすることで、さまざまなネットワーク復活サービスによるいつの間に通信コンテンツの保存を支援し、任天堂がサービスを停止した後も、これらのタイトルのいつの間に通信機能を利用できるようにすることができます。データベースのバックアップは収集時に匿名化されます。バックアップはネットワーク復活サービスに送られ、いつの間に通信機能を後で複製できるようにします。

Popular 3DS titles using SpotPass features include:

  • すれちがいMii広場: SpotPass was used to distribute special Miis (often of Nintendo employees) and puzzle pieces for the Puzzle Swap minigame. Without SpotPass, Puzzle Swap would normally not receive new puzzles.
  • Mario Kart 7: SpotPass was used to receive ghost data from other players and Nintendo.
  • Pokémon games: SpotPass was used to distribute special Pokémon during events.
  • New Super Mario Bros. 2: SpotPass was used to receive a certain system message allowing the installation of DLC.
  • Tomodachi Life: SpotPass was used to distribute special outfits and send messages to other islands.
  • Swapnote and Swapdoodle: SpotPass was used for sending and receiving notes (including notes from Nintendo).

FAQ

Q. Apart from dumping my system's SpotPass data, what else can I do to help?

A. Spread the word everywhere. We only have until April 8th to archive as much as we can, so make sure that people understand the urgency of the situation.


Q. Which games have SpotPass data archived?

A. Check the spreadsheets (this one for 3DS, this one for Wii U) to see which games currently have data archived. Do note that if a game is present or marked as "complete", that does NOT mean that all data for that game has been collected. Rather, it means that some data for the game has been collected and it may or may not actually be complete.


Q. How do I make sure that [game] has its SpotPass data archived?

A. Reach out to others who have played that game, as many as possible for a greater chance of yielding new data, and tell them about the archival project and how they can get their data preserved. There's a lot of games (especially niche games) that don't have SpotPass data archived yet, so it's very important for game communities to know about this in order to ensure that SpotPass data for their game(s) is preserved before it's too late. If you have any more specific questions about this, reach out to us on Discord.
